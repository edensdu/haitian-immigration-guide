# Immigration Guide for Haitians / Gid Imigrasyon pou Ayisyen

A simple, user-friendly web application to help Haitian immigrants understand their current immigration status, available options, timelines, and resources for achieving permanent residency in the United States.

## Features

- **Status Assessment**: Simple questionnaire to identify current immigration status
- **Status Information**: Detailed information about what each status means
- **Available Options**: Clear explanation of pathways available based on status
- **Timeline Visualization**: Visual timeline showing typical processing times
- **Resources**: Links to legal help, government websites, and community resources
- **Bilingual Support**: Available in both English and Haitian Creole (Krey√≤l)

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.
